

# 📖 UspSocDownloader - Руководство пользователя

> Полное руководство по использованию Telegram-бота для скачивания контента из социальных сетей

---

## 🚀 Начало работы

### Запуск бота

1. Найдите [@UspSocDownloader_bot](https://t.me/UspSocDownloader_bot) в Telegram
2. Нажмите `/start`
3. Отправьте ссылку на видео/фото из поддерживаемой социальной сети

### Поддерживаемые платформы

- 📸 **Instagram** - посты, reels, карусели
- 📺 **YouTube** - видео, shorts, только аудио
- 🎵 **TikTok** - видео
- 🐦 **Twitter/X** - видео, фото, несколько видео
- 🎬 **VK** - видео, clips

## 🎯 Основные функции

### 1. Скачивание контента

#### Как скачать видео/фото:
1. Скопируйте ссылку из соц. сети
2. Отправьте ссылку боту
3. Выберите качество (для YouTube)
4. Дождитесь загрузки

#### Поддерживаемые форматы ссылок:

**Instagram:**
```
https://www.instagram.com/p/ABC123/
https://instagram.com/reel/XYZ789/
```

**YouTube:**
```
https://youtube.com/watch?v=ABC123
https://youtu.be/ABC123
https://youtube.com/shorts/ABC123
```

**TikTok:**
```
https://tiktok.com/@user/video/123456
https://vm.tiktok.com/ABC123/
```

**Twitter/X:**
```
https://twitter.com/user/status/123456
https://x.com/user/status/123456
```

**VK:**
```
https://vk.com/video-12345_67890
https://vk.com/clip-12345_67890
```

---

### 2. История загрузок

Команда: `/history`

**Возможности:**
- Просмотр последних 10 загрузок
- Кнопка "Скачать снова" для повторной загрузки
- Добавление/удаление из избранного (⭐)
- Добавление в коллекцию (📁)
- Показать больше загрузок
- Поиск по истории (🔍)
- Экспорт истории (📤)

**Информация для каждой загрузки:**
- Иконка платформы (📸 📺 🎵)
- Название
- Дата и время
- Размер файла
- Статус избранного (⭐)

---

### 3. Поиск по истории

Команда: `/search`

**Способы поиска:**

#### Текстовый поиск
Введите любой текст для поиска в названиях, авторах и URL:
```
Введите: Paris
Результат: Все загрузки с "Paris" в названии
```

#### Быстрые фильтры:

**📱 По платформе:**
- Instagram
- YouTube
- TikTok
- Twitter
- VK

**📅 По дате:**
- Последние 7 дней
- Последний месяц

**🎬 По типу контента:**
- Видео
- Фото
- Аудио
- Карусель

**👤 По автору:**
Поиск по имени создателя контента

**Пример комбинированного поиска:**
1. Выберите "По платформе" → Instagram
2. В результатах ищите по тексту "sunset"
3. Получите все Instagram загрузки с "sunset"

---

### 4. Избранное

Команда: `/favorites`

**Как добавить в избранное:**
1. Откройте `/history`
2. Нажмите ⭐ рядом с загрузкой
3. Файл добавлен в избранное

**Как удалить из избранного:**
- Нажмите 💔 в `/history` или `/favorites`

**Особенности:**
- Избранные файлы не удаляются автоматически
- Можно экспортировать только избранное
- Быстрый доступ к любимым загрузкам

---

### 5. Коллекции

Команда: `/collections`

**Создание коллекции:**
1. Нажмите "➕ Создать коллекцию"
2. Введите название (можно с эмодзи)
3. Коллекция создана!

**Добавление в коллекцию:**
1. Откройте `/history`
2. Нажмите 📁 рядом с загрузкой
3. Выберите коллекцию
4. Готово!

**Редактирование коллекции:**
1. Откройте коллекцию
2. Нажмите "✏️ Редактировать"
3. Выберите что изменить:
   - ✏️ Название
   - 🎨 Иконка (эмодзи)

**Удаление коллекции:**
1. Откройте коллекцию
2. Нажмите "🗑️ Удалить"
3. Подтвердите удаление

---

### 6. Экспорт данных

Команда: `/export`

**Форматы экспорта:**

#### 📊 CSV (Excel, Google Sheets)
Идеально для анализа в таблицах:
```csv
id,date,platform,title,author,url,file_size_mb
1,2026-03-02,Instagram,Beautiful sunset,john_photo,https://...,5.2
```

#### 📋 JSON (Программирование)
Структурированный формат для разработчиков:
```json
{
  "id": 1,
  "platform": "Instagram",
  "title": "Beautiful sunset",
  "file_size_mb": 5.2
}
```

**Варианты экспорта:**
- Вся история
- Только избранное
- По конкретной платформе

**Что включается:**
- ID, дата, платформа
- Название, автор
- Ссылка на оригинал
- Размер файла
- Статус избранного
- Коллекция (если добавлено)

---

### 7. Настройки

Команда: `/settings`

**Доступные настройки:**

#### 🎬 Качество по умолчанию
Выберите предпочитаемое качество для YouTube:
- 360p - Низкое (экономия трафика)
- 480p - Среднее
- 720p - HD (по умолчанию)
- 1080p - Full HD (только Premium)

#### 📁 Формат видео
- MP4 (по умолчанию, универсальный)
- WebM (меньший размер)

#### 🗑️ Авто-удаление файлов
Срок хранения скачанных файлов:
- 3 дня
- 7 дней (по умолчанию)
- 14 дней
- 30 дней
- Никогда

**⚠️ Важно:** Избранные файлы (⭐) не удаляются автоматически!

#### 🔔 Уведомления
Включить/выключить уведомления от бота

## 🤖 AI-функции

### Перевод текста

**Где доступно:**
После скачивания контента с текстом

**Как использовать:**
1. Скачайте пост с текстом
2. Нажмите кнопку "🇷🇺 Перевести"
3. Получите перевод на русский

### Рерайт текста

**Доступные стили:**
- 📚 Экспертный - профессиональный стиль
- 😄 Юмористический - с шутками
- 👋 Дружелюбный - неформальный стиль

**Как использовать:**
1. Скачайте пост с текстом
2. Нажмите "✍️ Рерайт"
3. Выберите стиль
4. Получите переписанный текст

### OCR (Распознавание текста)

**Назначение:**
Извлечение текста с изображений

**Как использовать:**
1. Скачайте фото с текстом
2. Нажмите "👁️ Распознать текст"
3. Получите извлеченный текст

   - 720p достаточно для большинства случаев

## ❓ Частые вопросы (FAQ)

### Скачивание

**Q: Почему не скачивается Instagram?**
A: Проверьте:
- Ссылка корректная
- Пост публичный (не приватный аккаунт)
- Пост не удален

**Q: YouTube видео не качается**
A: Возможные причины:
- Видео с возрастными ограничениями
- Прямая трансляция (дождитесь окончания)
- Premium-контент

**Q: Файл слишком большой**
A: Telegram лимит - 50 MB для видео:
- Выберите меньшее качество
- Скачайте только аудио (YouTube)

### История и поиск

**Q: Где найти старую загрузку?**
A: Используйте поиск `/search`:
- Введите часть названия
- Фильтр по платформе
- Фильтр по дате

**Q: Как удалить из истории?**
A: История не удаляется, но:
- Файлы удаляются автоматически
- Можно удалить из коллекции
- Можно убрать из избранного

### Коллекции

**Q: Сколько коллекций можно создать?**
A: Без ограничений

**Q: Можно ли добавить в несколько коллекций?**
A: Нет, файл может быть только в одной коллекции

**Q: Что происходит при удалении коллекции?**
A: Коллекция удаляется, файлы остаются в истории

### Экспорт

**Q: Какой формат выбрать?**
A:
- CSV - для Excel/Google Sheets
- JSON - для программирования

**Q: Включаются ли файлы в экспорт?**
A: Нет, только метаданные и ссылки
