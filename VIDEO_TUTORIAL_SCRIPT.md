# 🎥 Видео-туториал для UspSocDownloader

> Скрипт для создания обучающего видео для пользователей

---

## 📋 Информация о видео

**Продолжительность:** 5-7 минут
**Формат:** Screencast с голосовым комментарием
**Язык:** Русский
**Целевая аудитория:** Новые пользователи бота

---

## 🎬 Сцены

### Сцена 1: Введение (30 секунд)

**Что показываем:**
- Telegram открыт на экране
- Бот [@UspSocDownloader_bot](https://t.me/UspSocDownloader_bot)

**Текст:**
```
Привет! В этом видео я покажу, как пользоваться UspSocDownloader -
ботом для скачивания видео и фото из Instagram, YouTube, TikTok и других
социальных сетей.

Бот полностью бесплатный и очень простой в использовании.
```

**Действия:**
1. Показать иконку бота
2. Нажать "Запустить" / Start

---

### Сцена 2: Базовое скачивание (1.5 минуты)

**Что показываем:**
- Скачивание Instagram Reels
- Скачивание YouTube видео
- Скачивание TikTok видео

**Текст:**
```
Использовать бот очень просто - достаточно отправить ему ссылку на контент.

Давайте попробуем скачать Instagram Reels.
[Копируем ссылку из Instagram]
[Вставляем в бота]

Бот автоматически определяет платформу и начинает скачивание.
Через несколько секунд файл готов!

Точно так же работает с YouTube...
[Показываем скачивание YouTube видео]

...и с TikTok.
[Показываем скачивание TikTok видео]
```

**Действия:**
1. Открыть Instagram в браузере
2. Скопировать ссылку на Reels
3. Вставить в бота
4. Дождаться загрузки
5. Повторить для YouTube и TikTok

---

### Сцена 3: Выбор качества (1 минута)

**Что показываем:**
- YouTube с выбором качества
- Скачивание только аудио

**Текст:**
```
Для YouTube вы можете выбрать качество видео.

Отправляем ссылку на YouTube видео...
Бот предлагает выбрать качество: 360p, 480p, 720p или 1080p.

Также можно скачать только аудио в формате MP3 -
отлично подходит для музыки и подкастов.
```

**Действия:**
1. Отправить YouTube ссылку
2. Показать меню выбора качества
3. Выбрать 720p
4. Показать опцию "Только аудио (MP3)"
5. Скачать аудио

---

### Сцена 4: AI-функции (1.5 минуты)

**Что показываем:**
- Перевод текста
- Рерайт в разных стилях
- OCR с изображений

**Текст:**
```
У бота есть интересные AI-функции на основе GPT-4.

После скачивания поста с текстом появляются кнопки:

1. "Перевести на русский" - автоматический перевод любого текста.
[Показываем перевод английского текста]

2. "Сделать рерайт" - переписывает текст в разных стилях:
   - Экспертный стиль - для серьезных публикаций
   - Юмористический - добавляет шутки
   - Дружелюбный - более личный тон
[Показываем рерайт в юмористическом стиле]

3. "Получить текст с картинок" - извлекает текст с изображений.
   Полезно для скриншотов и инфографики.
[Показываем OCR на карусели с текстом]
```

**Действия:**
1. Скачать Instagram пост с английским текстом
2. Нажать "Перевести на русский"
3. Показать результат
4. Нажать "Сделать рерайт" → "Юмористический стиль"
5. Показать результат
6. Скачать карусель с текстом на изображениях
7. Нажать "Получить текст с картинок"
8. Показать результат

---

### Сцена 5: История и избранное (1 минута)

**Что показываем:**
- Команда /history
- Повторное скачивание
- Добавление в избранное
- Команда /favorites

**Текст:**
```
Все ваши загрузки сохраняются в истории.

Команда /history показывает последние загрузки.
Вы можете скачать любой файл повторно - без повторной загрузки!

Понравившиеся загрузки можно добавить в избранное.
[Показываем добавление в избранное]

Команда /favorites показывает все избранное.
```

**Действия:**
1. Отправить `/history`
2. Показать список загрузок
3. Нажать "Скачать снова" на одной из загрузок
4. Нажать кнопку "⭐" (добавить в избранное)
5. Отправить `/favorites`
6. Показать избранное

---

### Сцена 6: Коллекции (1 минута)

**Что показываем:**
- Создание коллекции
- Добавление файлов в коллекцию
- Просмотр коллекции

**Текст:**
```
Для организации загрузок есть коллекции.

Команда /collections показывает все ваши коллекции.
[Показываем команду]

Создадим новую коллекцию "Мои видео".
[Создаем коллекцию]

Теперь после каждой загрузки можно добавлять файл в коллекцию.
[Показываем добавление в коллекцию]

В любой момент можно открыть коллекцию и посмотреть все файлы.
```

**Действия:**
1. Отправить `/collections`
2. Нажать "Создать коллекцию"
3. Ввести название "Мои видео"
4. Выбрать иконку 🎬
5. Скачать видео
6. Нажать "📁 В коллекцию"
7. Выбрать "Мои видео"
8. Открыть `/collections` → "Мои видео"
9. Показать содержимое

---

### Сцена 7: Настройки (45 секунд)

**Что показываем:**
- Команда /settings
- Изменение качества по умолчанию
- Настройка авто-удаления

**Текст:**
```
В настройках можно настроить бота под себя.

Команда /settings открывает меню настроек:
- Качество по умолчанию - чтобы не выбирать каждый раз
- Формат файлов
- Авто-удаление старых файлов через N дней

Например, установим качество 1080p по умолчанию.
[Показываем изменение настройки]

Теперь YouTube видео будут скачиваться в 1080p автоматически.
```

**Действия:**
1. Отправить `/settings`
2. Показать меню настроек
3. Нажать "🎬 Качество: 720p"
4. Выбрать "1080p"
5. Показать обновленные настройки

### Титры начальные
```
UspSocDownloader
Скачивай контент из соцсетей легко

Instagram • YouTube • TikTok • Twitter • VK
```

### Титры финальные
```
@UspSocDownloader_bot

Больше информации:
GitHub: github.com/ircitdev/UspSocDownloader
Документация: см. USER_GUIDE.md

Спасибо за просмотр!
```

### Музыка
- Фоновая музыка: легкая, ненавязчивая
- Громкость: тихая, чтобы не мешать голосу
- Рекомендации: YouTube Audio Library (No Copyright Music)

### Визуальные эффекты
- Подсветка курсора
- Увеличение важных элементов (zoom)
- Стрелки/аннотации для пояснения
- Плавные переходы между сценами

## 📝 Подготовка к записи

### Технические требования
- **Разрешение:** 1920x1080 (Full HD)
- **FPS:** 30 или 60
- **Программа для записи:** OBS Studio / Camtasia / ScreenFlow
- **Микрофон:** Хорошее качество звука
- **Telegram:** Desktop версия

### Перед записью
1. Очистить Telegram от личных чатов (privacy)
2. Создать тестовый аккаунт или использовать режим "чистого" интерфейса
3. Подготовить все ссылки заранее
4. Проверить работу бота
5. Написать черновик текста
6. Сделать несколько пробных записей

### Тестовые ссылки для записи

**Instagram:**
```
https://www.instagram.com/p/[публичный_reels]
https://www.instagram.com/p/[карусель_с_фото]
```

**YouTube:**
```
https://www.youtube.com/watch?v=dQw4w9WgXcQ
https://www.youtube.com/shorts/[популярный_shorts]
```

**TikTok:**
```
https://www.tiktok.com/@[популярный_аккаунт]/video/[ID]
```

**Twitter:**
```
https://twitter.com/[аккаунт]/status/[ID]
```
